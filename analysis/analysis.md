Data cleaning and feature engineering were performed before model training.

## 4. Model â€” Linear Regression

### Theory

Linear Regression assumes a **linear relationship** between the independent variables and the target output.
In the context of solar power generation, it provides a good **baseline model** to understand the trend and relative importance of features like irradiation and temperature.

### Code

```python
from sklearn.linear_model import LinearRegression
from sklearn.metrics import r2_score, mean_squared_error

linreg_dc = LinearRegression()
linreg_ac = LinearRegression()

linreg_dc.fit(X_train, y_train_dc)
linreg_ac.fit(X_train, y_train_ac)

y_pred_dc = linreg_dc.predict(X_test)
y_pred_ac = linreg_ac.predict(X_test)

r2_dc = r2_score(y_test_dc, y_pred_dc)
r2_ac = r2_score(y_test_ac, y_pred_ac)

print("DC Power RÂ²:", r2_dc)
print("AC Power RÂ²:", r2_ac)
```
